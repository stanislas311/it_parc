IT Parc — Gestion du Parc Informatique

TECHPARK CI — Direction des Systèmes d'Information — Abidjan

Module complet de gestion du parc informatique : équipements, affectations, interventions, contrats fournisseurs, alertes automatiques, rapports PDF, exports Excel et dashboard OWL.

Fonctionnalités clés

  • Workflow équipement : Brouillon → Affecté → Maintenance → Retiré
  • Wizard de réaffectation avec historique et motif
  • Interventions corrective/préventive avec vue calendrier
  • Alertes garanties et contrats (cron + scan manuel)
  • Import CSV en masse
  • Dashboard OWL avec KPIs en temps réel
